# createEditorMenuItem

## Argument

- content : Require
  - string
- elemBefore
  - ReactComponents
- onClick
  - (editorActions: EditorActions) => void

## Default usage

```jsx
createEditorMenuItem({ content: "Content Name" });
```

### Return

```jsx
{
  content: "Content Name",
  value: { name: "contentName" },
  tooltipDescription: "Content Name",
  tooltipPosition: "right",
  elemBefore: <CustomSVG width="24" height="24" d={TEST_ICON} />,
  onClick: null
}
```

## onClick Event

```jsx
createEditorMenuItem({
  content: "Content Name",
  onClick: () => alert("Click Button")
});
```

### Return

```jsx
{
  content: "Content Name",
  value: { name: "contentName" },
  tooltipDescription: "Content Name",
  tooltipPosition: "right",
  elemBefore: <CustomSVG width="24" height="24" d={TEST_ICON} />,
  onClick: () => alert("Click Button")
}
```

## Custom Button Element

- `elemBefore` 인자가 svg가 아닐 경우 정상적으로 렌더링되지 않을 수 있습니다.
  - svg를 쉽게 작성하기 위해 [CustomSVG Component](/docz-basics-custom-svg)를 제공합니다.

```jsx
createEditorMenuItem({
  content: "Content Name",
  elemBefore: <div>Button</div>
});
```

### Return

```jsx
{
  content: "Content Name",
  value: { name: "contentName" },
  tooltipDescription: "Content Name",
  tooltipPosition: "right",
  elemBefore: <div>Button</div>,
  onClick: null
}
```
